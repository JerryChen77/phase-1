#### 2、写出定义方法的语法中每一部分的含义

```
public static void main(String[] args){
	syso
}
访问修饰符 静态 返回值类型 方法名 (形参列表){
	方法体
}
```

#### 3、解释一下概念： 

- 形参是什么？	
  - 形式参数
  - 定义在方法声明上，在方法名后面的小括号中
  - 表示调用此方法需要传入对应类型的数据
- 实参是什么？
  - 调用方法时候传入的数据
- 形参和实参之间实现参数传递要注意哪些事项？
  - 数量相同
  - 类型相同
  - 顺序相同
